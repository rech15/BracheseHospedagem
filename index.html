Branches

São ramificações de um projeto de software ou documento e tem o objetivo o desenvolvimento paralelo de:

  • Novas funcionalidades
  • Correção de problemas
  • Alteração de código
Branch Master ou Main
Braço principal de um projeto em um repositório.
Branch Secundária
Braço secundário de um projeto em um repositório.

Hospedagem

O GitHub Pages oferece hospedagem gratuita de sites. Basta criar uma conta e um repositório público.
